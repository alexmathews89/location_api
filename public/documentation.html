This project will serve as the API and business logic layer for the Seattle loccations web app where a user can create an account, login/authenticate themselves, update their profile and get information about various locations in the city of Seattle. This web app will use express to make its server requests

Endpoints:

Url Endpoint Method Description Format Request Example Response Example
/locations GET Gets a list of all locations None None An array of JSON objects holding data about each location
/users POST Allows a user to register for an account JSON {username: "", password: "", email: "", birthday: ""} {Username: "chosen name", UserID: "assigned user ID", Password: "chosen password", Email: "chosen email", Birthday: "chosen birthday", FavoriteLocations: []}
/locations/:title GET Gets a location by its title None None {"Title": "", "Description": "", "LocatedAt": {"FromStadiums": ""}, "DateNamed": ""}
/locations/locatedat/:direction GET Gets a location based on where it is at in relation to the stadiums downtown None None {"FromStadiums": ""}
/users/:id PUT Allows a user to update thier information JSON { "Username": "Updated user name" } {"Username": "Updated user name", "UserID": "assigned userID", "FavoriteLocations": [] }
/users/:id/:locationTitle POST Adds a location to a users list of favorites None None A JSON object containing a user's updated array of favorite locations
/users/:id/:locationTitle DELETE Removes a location from a users list of favorites None None A JSON object containing a user's updated array of favorite locations
/users/:id DELETE Allows a user to delete their account None None None
